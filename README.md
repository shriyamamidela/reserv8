
# ReserV8 ‚Äì A Dynamic, Multi-purpose Reservation System

ReserV8 is a web-based reservation platform built to manage **restaurant table bookings** and **campus facility reservations** (such as library study rooms and sports grounds) at **Mahindra University**. It provides a clean, simple interface to make and view reservations.

### 3. Set Up the MySQL Database

* Open MySQL and run the provided SQL script (`schema.sql` or similar) to create the necessary tables.
* The following SQL queries are used to set up your database:

```sql
-- Create Database
CREATE DATABASE IF NOT EXISTS reserv8_db;
USE reserv8_db;

-- Drop tables if they exist
DROP TABLE IF EXISTS table_availability;
DROP TABLE IF EXISTS users;
DROP TABLE IF EXISTS restaurants;

-- Create Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    email VARCHAR(255) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Create Restaurants Table
CREATE TABLE restaurants (
    Name VARCHAR(49),
    Links VARCHAR(88),
    image_url VARCHAR(1500),
    Cost INT,
    Collections VARCHAR(142),
    Cuisines VARCHAR(65),
    Timings VARCHAR(168),
    Restaurant VARCHAR(49),
    Reviewer VARCHAR(24),
    Review VARCHAR(1426),
    Rating INT,
    Latitude DECIMAL(9,7),
    Longitude DECIMAL(9,7),
    Address VARCHAR(227),
    PRIMARY KEY (Restaurant)
);

-- Create Table Availability Table
CREATE TABLE table_availability (
    id INT AUTO_INCREMENT PRIMARY KEY,
    restaurant_id VARCHAR(255),
    date DATE NOT NULL,
    time_slot TIME NOT NULL,
    total_tables INT DEFAULT 10,
    booked_tables INT DEFAULT 0,
    FOREIGN KEY (restaurant_id) REFERENCES restaurants(Restaurant)
);
```
